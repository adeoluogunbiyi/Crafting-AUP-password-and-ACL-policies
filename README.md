# Crafting-AUP-password-and-ACL-policies
## Overview
Strong security begins with clear, enforceable policies. In any enterprise environment—or even in a personal cybersecurity lab—three foundational documents shape how users interact with systems and how access is controlled: the Acceptable Use Policy (AUP), the Password Policy, and Access Control Lists (ACLs). Each plays a distinct role, but together they create a layered defense that reduces risk and improves accountability.
